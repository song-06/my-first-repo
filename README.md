## 1. top
- 역할: 시스템에서 실행 중인 프로세스들의 실시간 정보를 보여주는 명령어
- 특징:
  - 1  메모리 사용량, cpu, 프로세스 상태를 실시간으로 모니터링할 수 있음
  - 2  시스템 성분 분석
 

- ***-u 사용자명: 특정 사용자 프로세스만 표시함***
- ***-p PID: 특정 프로세스 ID만 표시***


> 예시 )
>> top
>>> 실행 후, q를 눌러 종료 가능
***

## 2. ps
- 역할: 현재 시스템에서 실행 중인 프로세스 목록을 출력
- 특징:
  - 1  실시간 갱신X, 명령 실행 시점의 스냅샷만 표시함
  - 2  커스텀 출력이 가능함
 

- ***ps aux: 모든 사용자의 모든 프로세스 출력***
- ***ps -ef: 포맷을 바꿔 상세 정보 출력***


> 예시 )
>> ps aux | grep firefox
>>> 현재 실행 중인 firefox 프로세스 확인
***

## 3.job
- 역할: 현재 쉘 세션에서 백그라운드,중지된 작업 목록 확인함
- 특징:
  - 1  쉘에서 시작한 작업에 대해서만 상태 확인 가능함
  - 2  백그라운드/포그라운드 전환과 관련 있음
 

- ***jobs -l: PID 포함한 상세 정보 표시***
- ***jobs -p: PID만 출력***


> 예시 )
>> sleep 100 &
>>
>> 
>> jobs
>>> 백그라운드에서 실행 중인 sleep 100 작업 확인
***

## 4.kill
- 역할: 프로세스에 **신호(signal)**를 보내 종료하거나 제어
- 특징:
  - 1  kill로 프로세스 종료 가능, kill -l로 사용 가능한 신호 확인 가능
  - 2  가장 흔히 사용하는 신호는 SIGKILL(9)과 SIGTERM(15)
 

- ***kill PID: 기본 신호(SIGTERM) 보내기***
- ***kill -9 PID: 강제 종료(SIGKILL)***


> 예시 )
>> kill 12345
>>
>> 
>> kill -9 12345
***
